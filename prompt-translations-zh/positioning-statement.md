# 产品定位声明模板

<!--
## 描述：
此模板提供了为产品或服务制定定位声明的结构化方法，遵循 Geoffrey Moore 在《跨越鸿沟》中建议的框架。定位声明对于简洁地传达目标客户是谁、他们未满足的需求、产品或服务的类别、其关键好处以及它如何与竞争对手或竞争格局区分开来至关重要。

## 使用说明：
在生成式 AI 会话中使用此模板之前，请确保会话已预加载以下背景信息：
    - 特定的目标客户群及其定义特征。
    - 目标客户未满足或服务不足的需求。
    - 产品或服务如何在其类别中满足这些需求。
    - 使产品或服务与众不同的关键好处和价值主张。
    - 可选但重要的是对竞争格局的理解，包括直接竞争对手和更广泛的市场挑战（提示：你可以使用 Bing/Copilot 捕获此背景并将其粘贴到你的会话中）。

## 使用说明：
1. 确保上述背景已在你的会话中明确建立。
2. 使用以下提示词来指导创建定位声明：

"根据提供的背景，请使用以下定位声明模板创建定位声明，以 Markdown 代码块的形式呈现。"

```markdown
## 定位声明模板

<!-- 使用以下模板生成 Geoffrey Moore 风格的定位声明 -->

### 价值主张

**为了** [目标客户/用户（考虑用户角色）]
- **需要** [用户未被满足需求的声明（考虑痛点、收益、待完成任务）]
- [产品或服务的名称（如果未提供，创建一个引人注目的产品名称）]
- **是一个** [产品类别的定义]
- **它** [对用户的好处声明（关注结果，而非功能）]。

### 差异化声明

- **不像** [主要竞争对手或竞争领域/市场/舞台]，
- [产品或服务的名称]
- **提供** [产品或服务如何独特地差异化（再次关注结果，而非功能）]。
```
