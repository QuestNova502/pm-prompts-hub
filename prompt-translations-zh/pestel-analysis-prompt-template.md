# PESTEL 分析提示词模板

## 概述

- **项目/产品名称**：[正在分析的项目或产品的名称。]
- **分析目的**：[进行此 PESTEL 分析的具体目标或原因。]
- **分析师**：[进行分析的个人或团队的姓名。]
- **日期**：[进行分析的日期。]

## 1. 政治因素

- **政府政策**：[识别并分析政府政策如何影响项目/产品。]
- **政治稳定性**：[评估与项目/产品相关的地区的政治稳定性的影响。]
- **贸易法规**：[检查贸易法规及其对项目/产品的潜在影响。]
- **税收政策**：[分析税收政策及其对项目/产品的影响。]

## 2. 经济因素

- **经济增长**：[评估经济增长率及其对项目/产品的潜在影响。]
- **通货膨胀率**：[考虑通货膨胀率及其对定价和成本的影响。]
- **汇率**：[分析汇率波动如何影响项目/产品。]
- **消费者支出**：[评估消费者支出水平及其对项目/产品的影响。]

## 3. 社会因素

- **人口统计**：[检查人口统计及其如何影响项目/产品的市场。]
- **文化趋势**：[分析当前的文化趋势及其对需求的潜在影响。]
- **生活方式变化**：[考虑生活方式变化及其对项目/产品的影响。]
- **消费者态度**：[评估消费者态度和行为以及它们如何影响项目/产品。]

## 4. 技术因素

- **技术进步**：[识别技术进步及其影响项目/产品的潜力。]
- **研发活动**：[评估该部门的研发水平及其影响。]
- **自动化**：[评估自动化的作用及其对效率和成本的影响。]
- **数字化转型**：[考虑数字化转型趋势如何影响项目/产品。]

## 5. 环境因素

- **气候变化**：[分析气候变化对项目/产品的影响。]
- **可持续性实践**：[评估可持续性实践对运营和市场定位的影响。]
- **资源稀缺**：[评估资源稀缺的风险和影响。]
- **环境法规**：[检查环境法规及其对项目/产品的潜在影响。]

## 6. 法律因素

- **合规要求**：[识别与项目/产品相关的法律和合规要求。]
- **知识产权**：[评估知识产权及其保护的重要性。]
- **劳动法**：[考虑劳动法及其对劳动实践的影响。]
- **健康和安全法规**：[评估健康和安全法规及其对运营的影响。]
